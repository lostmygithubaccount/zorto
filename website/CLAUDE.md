# zorto.dev

Project site for [zorto](https://github.com/dkdc-io/zorto). Built with zorto.

## Commands

- `bin/preview.sh`: Start dev server with auto-open browser
- `zorto build`: Build site to `public/`

## Styling

Dark mode default with light mode toggle. Color scheme: violet/purple neon + cyan accents. Same theme as dkdc.io / dkdc.sh.

## Content style

- Use sentence casing for headings (capitalize first word only, unlike dkdc's all-lowercase)
- Use "Zorto" (capitalized) in prose; lowercase `zorto` for commands/code/URLs
