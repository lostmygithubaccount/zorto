The `zorto` command-line interface. Install from [crates.io](https://crates.io/crates/zorto) or [PyPI](https://pypi.org/project/zorto/). API docs at [docs.rs](https://docs.rs/zorto-core).

## zorto

```{bash}
zorto --help
```

```{bash}
zorto preview --help
```

```{bash}
zorto init --help
```

```{bash}
zorto check --help
```
