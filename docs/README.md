# Zorto documentation

Zorto is a static site generator with executable code blocks, written in Rust with Python bindings.

```bash
zorto init my-site && cd my-site && zorto preview
```

## Documentation

- [Getting started](getting-started/README.md): install, create, and preview a site in minutes
- [Concepts](concepts/README.md): content model, templates, themes, configuration, executable code
- [How-to](how-to/README.md): step-by-step guides for blogging, deployment, SEO, and more
