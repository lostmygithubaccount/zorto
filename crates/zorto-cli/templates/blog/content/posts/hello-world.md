## Getting started

Each post is a Markdown file with a TOML frontmatter header (the `+++` block above). The frontmatter sets the title, date, and description.

### Executable code blocks

Zorto supports executable code blocks. Try adding a Python block:

```python
print("Hello from Zorto!")
```

### What's next

- Edit this post or create new ones in `content/posts/`
- Customize the templates in `templates/`
- Run `zorto preview --open` to see your changes live
